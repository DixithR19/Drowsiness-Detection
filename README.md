# Drowsiness-Detection

1. Python 2.7+
2. face_utils
3. dlib
4. cv2
5. playsound

>https://github.com/AKSHAYUBHAT/TensorFace/blob/master/openface/models/dlib/shape_predictor_68_face_landmarks.dat

### Usage
```
python Drowsiness_Detection.py
```
